# JENSON-PROJECT--MySQL

Jenkins SQL Project:Database Automation & Analytics

This project demonstrates the automation of database testing, integration, and deployment workflows using Jenkins pipelines. It focuses on managing continuous data workflows and performing deep-dive SQL analytics for a bicycle and motorbike manufacturing business.

🚀 Project Overview
Purpose: Automate SQL workflows for continuous data integration.
Domain: Manufacturing (Bicycles & Motorbikes).
Key Tech: Jenkins, SQL (PostgreSQL/MySQL), Data Pipeline Management.

📊 Key SQL Analytics & Insights
The repository includes scripts to solve complex business queries, such as:
Sales Performance: Total products sold per store and identifying top-selling items by quantity.
Customer Insights: Identifying high-spending customers and those who purchased across all product categories.
Operational Efficiency: Identifying staff members with no sales records to optimize team performance.
Advanced Metrics: Calculating cumulative sums of quantities sold over time and median price list values.
